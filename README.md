This repository is intended to fulfill the company recruitment test by Elvan Fiernes Ernando.

Tech stack:

- React JS


## Installation

To install this project run this command:

Clone the project

```bash
  git clone https://github.com/elvanfiernesernando/bk_clone.git
```

Go to the project directory

```bash
  cd bk_clone
```

Install client dependencies

```bash
  npm install
```

Run project

```bash
  npm run start

  // client is running on port 3000
```
